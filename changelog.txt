『 Charger Protector Miyabi Core v2.0.0 』

v2.0.0
- Fixed charging disconnect-connect loop issue on Snapdragon chipsets (Poco M3).
- Optimized charger status detection using physical connection (present) and USB voltage sensors.
- Improved replug logic to prevent temporary charging upon cable reconnection (direct bypass).
- Implemented instant settings application (SIGUSR1 signal) without daemon loop delays.
- Center-aligned the CLI Dashboard title for better presentation.
- Shortened the Active Node display using its base filename to prevent text wrapping.
- Fixed Plugged In status indicator to display red (No) when protection is active on MediaTek devices.
